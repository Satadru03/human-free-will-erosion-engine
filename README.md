# human-free-will-erosion-engine

The 4 Core Metrics 

#### 1. **Decision Entropy**

* Shannon entropy over action distribution
* Low entropy = habit prison

#### 2. **Sequence Predictability**

* N-gram / Markov transition confidence
* “Given last N actions, what happens next?”

#### 3. **Autopilot Score**

* % of decisions matching dominant patterns

#### 4. **Free Will Index (0–100)**

```text
FWI = 100 − weighted_predictability − habit_strength
```
